# Job Listing App

**Live Demo:** [View the Dashboard](https://archana-lohana.github.io/job-listing-app/)


A modern **Job Listing Application** built using **React** and **Vite**.  
This project displays multiple job cards dynamically with reusable components, providing a clean and responsive user interface.

## Features

- **Reusable Job Cards**: Each card shows company logo, job title, pay, location, job type, and experience level.
- **Dynamic Listing**: Jobs are displayed from an array of objects, making it easy to scale.
- **Responsive Design**: Works on mobile and desktop screens.
- **Modern UI**: Clean layout with interactive buttons for "Apply" and "Save".

## Tech Stack

- **Frontend**: React
- **Bundler**: Vite
- **Styling**: CSS
- **Icons**: Lucide-React
